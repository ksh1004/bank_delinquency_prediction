## 2단계 — EDA (탐색적 데이터 분석)

**노트북:** `notebooks/02_eda.ipynb`

### 주요 발견

#### NPL비율 시계열 패턴
| 구간 | 추세 | 범위 |
|------|------|------|
| 2011Q4~2016Q1 | 상승 (고NPL) | 1.36% → 1.87% |
| 2016Q1~2022Q3 | 지속 하락 | 1.87% → 0.38% |
| 2022Q3~2025Q4 | 반등 | 0.38% → 0.59% |

#### 상관관계 분석 (NPL비율과의 Pearson r)
| 피처 | 상관계수 | 방향 |
|------|---------|------|
| 가계대출잔액 | 0.931 | 음(-) |
| BIS비율 | 0.891 | 음(-) |
| ROA | 0.593 | 음(-) |
| NIM | 0.531 | 양(+) |
| 실업률 | 0.287 | 양(+) |
| 기준금리 | 0.069 | 음(-) |

> **핵심 인사이트:** 기준금리의 직접 상관이 의외로 약함. 가계대출잔액·BIS비율이 훨씬 강한 신호.

#### Lag 상관관계 분석
- 기준금리·대출금리는 Lag 1~4 모두 |r| < 0.18 (시차 효과 약함)
- 가계대출잔액은 Lag 1~4 모두 |r| ≈ 0.94~0.95 (매우 강한 음의 상관 유지)

## 3단계 — ML 모델링 v1 (거시지표만, 절대값 예측)

**노트북:** `notebooks/03_modeling.ipynb`

### 설정
- **피처:** 거시경제 지표만 (현재값 + lag1 + diff1 → 15개)
- **타겟:** 다음 분기 NPL비율 절대값
- **분할:** Train 2011Q4~2022Q4 (45개) / Test 2023Q1~2025Q3 (11개)

### 결과
| 모델 | RMSE(Test) | R²(Test) |
|------|-----------|---------|
| Random Forest | 0.0761% | -0.75 |
| XGBoost | 0.1106% | -2.70 |
| LightGBM | 0.1959% | -10.59 |
| 선형 모델들 | 0.5~0.9% | 매우 낮음 |

### 문제점
- 모든 모델 R² 음수 → 단순 평균 예측보다 못함
- **원인:** 훈련 기간(하락 추세) vs 테스트 기간(반등 추세)의 패턴 전환을 모델이 학습 못함

## 5단계 — ML 모델링 v3 (NPL 변화량 예측) ★ 핵심

**노트북:** `notebooks/05_modeling_v3.ipynb`

### 핵심 아이디어 전환
- **기존:** 다음 분기 NPL 절대값 예측 → 추세 전환에 취약
- **변경:** 다음 분기 NPL **변화량(+/-)** 예측 → 방향성 포착에 유리
- 최종 NPL = `현재 NPL + 예측 변화량`으로 복원

### 피처 구성 (18개)
| 유형 | 피처 | 개수 |
|------|------|------|
| 거시지표 현재값 | base_rate, deposit_rate, loan_rate, unemployment_rate, household_loan_balance | 5 |
| 거시지표 Lag1 | 위 5개의 1분기 전 값 | 5 |
| 거시지표 Diff1 | 위 5개의 전 분기 대비 변화량 | 5 |
| NPL 자기회귀 | NPL_lag1, NPL_diff1, NPL_diff2 | 3 |

### 결과
| 모델 | RMSE(Test) | R²(Test) | MAPE |
|------|-----------|---------|------|
| **LightGBM** | **0.0171%** | **0.9112** | **2.37%** |
| Random Forest | 0.0344% | 0.6422 | 5.61% |
| XGBoost | 0.0646% | -0.26 | 9.97% |
| 선형 모델들 | 0.2~0.6% | 음수 | — |

### LightGBM 분기별 예측 결과
| 입력분기 | 실제 NPL | 예측 NPL | 오차 |
|---------|---------|---------|------|
| 2023Q1 | 0.41% | 0.408% | +0.002 |
| 2023Q4 | 0.50% | 0.496% | +0.004 |
| 2024Q4 | 0.59% | 0.557% | +0.033 |
| 2025Q1 | 0.59% | 0.588% | +0.002 |
| 2025Q3 | 0.57% | 0.568% | +0.002 |

**11분기 연속 방향(상승/하락) 예측 100% 정확 / 평균 절대 오차 0.0123%**

## 7단계 — SHAP 피처 영향력 분석

**노트북:** `notebooks/07_shap.ipynb`

### 분석 대상
- LightGBM (기본) — R²=0.91
- RandomForest (튜닝) — R²=0.86

### SHAP 중요도 순위 (LightGBM 기준, mean|SHAP|)
| 순위 | 피처 | SHAP 중요도 | 방향 |
|------|------|------------|------|
| 1 | 기준금리(1분기전) | 0.0367 | (+) |
| 2 | 실업률 | 0.0213 | (+) |
| 3 | 대출금리변화 | 0.0140 | (-) |
| 4 | 기준금리 | 0.0132 | (-) |
| 5 | NPL변화(2분기) | 0.0070 | (-) |

### 핵심 해석
1. **기준금리는 1분기 시차** — `기준금리(현재)` 4위, `기준금리(1분기전)` 1위. 금리 효과는 시차를 두고 나타남
2. **실업률의 비선형 기여** — 단순 상관계수(r=0.29)에서는 약해 보였으나, 다른 피처와 상호작용 속에서 2위
3. **NPL 자기회귀 피처는 상대적으로 낮은 순위** — v3에서 성능 향상의 핵심이었으나 SHAP에서는 낮게 나옴 (복잡한 상호작용으로 분산됨)
